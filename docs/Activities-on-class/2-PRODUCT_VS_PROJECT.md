# Hoạt động 2 - Product vs Project

| Nội dung | Project-based Software | Product-based Software |
|-----------|------------------------|------------------------|
| Requirement | Khách hàng quyết định | User quyết định |
| Mục tiêu | Hoàn thành dự án | Tăng trưởng sản phẩm |
| Growth | Không quan tâm | Rất quan trọng |
| User tham gia | Ít | Liên tục |
| Cải tiến liên tục | Không nhiều | Liên tục |
| Đo thành công | Đúng tiến độ | User retention |
